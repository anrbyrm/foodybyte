## How to Use

**Step 1:**

Download or clone this repository by using the link below:

```bash
https://github.com/anrbyrm/foodybyte.git
```

**Step 2:**

Go to project root and execute the following command in console to get the required dependencies:

```bash
flutter pub get
```
